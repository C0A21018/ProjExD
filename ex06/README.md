# 第6回
## ブロック崩し（ex06/block_crash.py）
### ゲーム概要
- ex06/block_crash.pyを実行すると，22x15個のブロックとボールが乗ったパドルが
600x600のスクリーンに描画され，パドルを動かしてブロックを壊すゲームが始まる
- ブロックを全て壊すことができたらクリア
- ボールが下に落ちてしまうとゲームオーバーで終了する
### 操作方法
- 左クリックでボールがパドルから射出される
- マウスを動かしてパドルを操作する
